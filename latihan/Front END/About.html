About Website

Tujuan sebenarnya website ini adalah untuk menyelesaikan tugas final dari dosen dengan Mata Kuliah WEB Development. dengan tema Pariwisata,saya kemudian memutuskan untuk membuat website untuk tempat wisata Lappa’-Laona ini.

Saya pribadi sudah pernah mengunjungi tempat ini dan merasa kalau tempat ini kedepannya bisa menjadi salah satu objek wisata andalan di Sulawesi Selatan.walaupun referensi tentang tempat wisata ini tergolong masih kurang,dan sangat sulit untuk dijangkau jadi ada baiknya website ini saya buat sekedar untuk memberi informasi terkait Lappa’-Laona dengan harapan para pengunjung website ini yang tadinya tidak tahu menjadi tahu dan berminat datang langsung ke lokasinya.

Saya berharap Website ini bermanfaat untuk Orang lain
